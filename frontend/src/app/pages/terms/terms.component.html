Terms / Policy

We prioritize data security and privacy for all schools using Bigezolite. This platform is free to use; you only pay for SMS credits when you send messages.

Data Security

  • Each school’s data is isolated and protected by access controls.
  • Authentication is secured with JWT; passwords are hashed.
  • All database operations use parameterized queries to prevent SQL injection.
  • SMS credentials are stored per school; only used for that school’s messaging.

Usage

  • The app is free; SMS charges apply based on your provider balance.
  • By using the app, you agree to use parent contact information responsibly.

Contact

For questions or requests about your data, contact apps@grealm.org.
